Como usar Date.now() para criar IDs para seus objetos

Date.now() retorna o número de milissegundos decorridos desde 1/1/1970, o que gera números grandes como 1722481709398. Estes números são excelentes para usar como ID de objetos em sistemas executados em uma única máquina.

Temos mais informações à respeito disso neste artigo no Notion e neste repositório.
